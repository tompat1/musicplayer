# Cover art

Add `.jpg`, `.jpeg`, `.png`, `.webp`, or `.avif` images here.

Use the same base name as the audio file when possible:

```txt
public/assets/audio/Stellar Pulse.mp3
public/assets/covers/Stellar Pulse.webp
```

Run `npm run catalog`, `npm run dev`, or `npm run build` to refresh `src/audioData.json`.
